# 📊 Basic Data Cleaner & Analyzer

A simple and automated Python script to load, clean, and visualize data. This project demonstrates foundational Data Analysis and Machine Learning preparation steps.

## 🚀 Features
- **Data Loading:** Creates and loads data using Pandas DataFrames.
- **Automated Cleaning:** Automatically detects missing values (`NaN`) and fills them using the mean (average) of the respective columns.
- **Data Visualization:** Generates a clean Bar Chart using Matplotlib to compare data points (e.g., Employee Salaries).

## 🛠️ Technologies Used
- **Python** - **Pandas** (Data Manipulation)
- **Matplotlib** (Data Visualization)
- **Jupyter Notebook** (Development Environment)

## 💡 How to Run
1. Clone this repository or download the `.ipynb` file.
2. Ensure you have Python and Jupyter installed (Miniconda is recommended).
3. Install required libraries if you haven't already: `pip install pandas matplotlib`
4. Open the notebook and run the cells sequentially.
